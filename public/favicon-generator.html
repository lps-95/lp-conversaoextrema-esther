Opção 1: Usar ferramenta online

  1. Acesse realfavicongenerator.net
  2. Faça upload do arquivo favicon.svg
  3. Baixe o pacote gerado
  4. Extraia os arquivos na pasta /public

Opção 2: Usar Canva/Figma

  1. Crie um design 512x512px
  2. Fundo escuro (#0F0A1E)
  3. Letra "E" dourada em Georgia Bold
  4. Exporte como PNG
  5. Use favicon.io para converter

Dica: O favicon.svg já está funcionando! Este arquivo é apenas para gerar versões .ico e .png se necessário.
