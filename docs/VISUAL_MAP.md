## ✨ Animações por Elemento

### Padrões de Entrada

```
SEÇÃO INTEIRA
  fade-in + slide-up
  Duração: 0.8s
  Delay: 0s

CARDS/BOXES
  slide-up
  Duração: 0.8s
  Delay: 0.1s, 0.2s, 0.3s... (defasado)

ESTATÍSTICAS/NÚMEROS
  scale-in + pulse-glow
  Duração: 2s (pulse contínuo)

CTA PRINCIPAL
  pulse-subtle (movimento micro)
  Duração: 3s infinito
```

### Interações (Hover)

```
CARDS
  border: texto-muted → button-primary
  background: opaco → mais opaco
  shadow: 0 → xl com cor de botão
  transform: scale 1 → 1.02
  transition: 300ms smooth

BOTÕES
  scale: 1 → 1.05
  shadow: forte
  cor de fundo: transição suave

LINKS
  color: text-tertiary → button-primary
  underline: implicit hover
```

## 📱 Responsividade

### Breakpoints Tailwind

```
Mobile (default)
  Container: full width
  Padding: px-4
  Text: sm/base

Tablet (sm: 640px)
  Layout: 1-2 cols onde aplicável
  Text: base/lg
  Padding: px-4

Desktop (md: 768px)
  Container: max-w-xl / max-w-4xl
  Layout: 2-3 cols
  Text: lg/xl
  Grid: grid-cols-2/3
```

```
VISITANTE CHEGA
    ↓
[HERO] - Proposta de Valor + 1ª CTA
    ↓
[PROBLEMA] - Diagnóstico (se identificou, continua)
    ↓
[OPORTUNIDADE] - Solução apresentada
    ↓
[PERSONAS] - "Isso é pra mim?" (Qualify)
    ↓
[PROVA SOCIAL] - Números que impressionam (Crédibilidade)
    ↓
[COMO FUNCIONA] - Entender o processo (Objeção: "Como?")
    ↓
[ANTES vs DEPOIS] - Visualizar transformação
    ↓
[TESTEMUNHO] - Prova social real (Objeção: "Funciona?")
    ↓
[GARANTIA] - Remover risco (Objeção: "E se não der certo?")
    ↓
[FAQ] - Responder dúvidas restantes
    ↓
[URGÊNCIA] - Spots limitados (Criar FOMO)
    ↓
[CTA FINAL] - Ação agora com deadline
    ↓
[FORMULÁRIO] → CRM/Email
    ↓
✅ LEAD CAPTURADO
```
