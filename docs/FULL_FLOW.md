## 📈 Fluxo de Conversão por Fase

| Fase        | Seção                        | Objetivo        | Métrica              |
| ----------- | ---------------------------- | --------------- | -------------------- |
| Capturar    | Hero                         | 1ª impressão    | Scroll down rate     |
| Diagnóstico | Problema + Personas          | Identificação   | Time on section      |
| Emocional   | Valor + Depoimentos + Origem | Vontade         | Emotional connection |
| Racional    | Stats + Processo             | Credibilidade   | Trust score          |
| Objeção     | Garantia + FAQ               | Segurança       | Remaining questions  |
| Urgência    | Urgência + CTA               | Ação agora      | Click through        |
| Trust       | Footer                       | Confiança final | Conversion           |

## 🎨 Visual Hierarchy

```
TAMANHO:  Hero > CTA > Seções > Cards > Meta
CORES:    Primary > Button > Secondary > Tertiary > Muted
WEIGHT:   Bold > Semibold > Regular > Light
SPACING:  Macro (seções) > Médio (cards) > Micro (texto)
```
