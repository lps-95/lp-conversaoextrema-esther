## 🛠️ Stack Técnico

### Frontend

- **Framework:** Next.js 16.0.10 (Turbopack)
- **Styling:** Tailwind CSS 3.4.10
- **Typography:** Playfair Display (headers) + Inter (body)
- **State:** React Hooks (useState)
- **Animations:** CSS Keyframes (8 tipos)

### Backend

- **API:** Next.js Serverless Functions (/api/submit)
- **Hosting:** Vercel (pronto para deploy)
- **Form Processing:** JSON + validation

### Analytics & Compliance

- **Analytics:** Plausible (privacy-focused)
- **Consent:** Custom banner (GDPR/LGPD ready)
- **Tracking:** Behavioral + Conversion events

### Performance Optimizations

- **Build Time:** 1.8s (Turbopack fast)
- **Static Generation:** All pages pre-rendered
- **Animations:** GPU-accelerated (transform + opacity)
- **Bundle:** +3KB total (acceptável)
- **Lazy Loading:** Built-in Next.js Image optimization

## 🎨 Sistema de Design

### Paleta Semântica

**Primary Colors**

- `primary-dark`: #000000 (backgrounds)
- `primary-light`: #1a1a1a (accents)

**Accent Colors**

- `accent-brown`: #8B7355 (elegância/premium)
- `accent-cream`: #F5E6D3 (suavidade)
- `accent-gold`: #D4AF37 (destaque/aspiração)
- `accent-silver`: #C0C0C0 (sofisticação)

**Semantic Text**

- `text-primary`: Button primary (títulos)
- `text-secondary`: Body text
- `text-tertiary`: Meta/hints
- `text-muted`: Disabled/footer

**Button States**

- `button-primary`: Black/call-to-action
- `button-primaryHover`: Darker on hover
- `button-secondary`: Alternative CTA
- `button-secondaryHover`: Hover state

### Animações (8 tipos)

```
1. fade-in (0.8s): Opacity 0→1
2. slide-up (0.8s): translateY(-20px)
3. scale-in (0.6s): scale(0.95)→(1)
4. pulse-glow (2s): Shadow pulse infinito
5. pulse-subtle (3s): Scale micro pulse
6. float-up (varies): Ondulante
7. shimmer (2s): Light sweep
8. glow-pulse (varies): Color pulse
```

### Typography Scale

```
Display (h1): 3xl-5xl Playfair Bold
Heading (h2): 2xl-3xl Playfair Bold
Subhead (h3): xl-2xl Playfair Bold
Body: lg Inter Regular
Small: sm-base Inter Regular
Tiny: xs Inter Regular
```
