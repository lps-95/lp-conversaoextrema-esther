## 🚀 Como Fazer Deploy (15 min)

### Quick Start

```bash
# 1. Preparar git
git init && git add . && git commit -m "Initial: v2.0"

# 2. Criar repo GitHub
# → github.com/new → Create repository

# 3. Conectar
git remote add origin https://github.com/seu-user/repo.git
git push -u origin main

# 4. Deploy Vercel
# → vercel.com → Import GitHub → Deploy ✅

# 5. Configurar env vars
NEXT_PUBLIC_PLAUSIBLE_DOMAIN=seu-dominio.com

# ✅ Live em ~2 minutos!
```
